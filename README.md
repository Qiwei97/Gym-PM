# Gym-PM 

This is a Gym RL Environment for Predictive Maintenance.

Compatible with gym-api, Stable Baselines3, RLlib.

### Set up

```
git clone https://github.com/Qiwei97/Gym-PM.git
cd Gym-PM
pip install -e .
```
